# Privacy

Phase 1 sends only system performance, session state, network totals/local address, and audio endpoint state. It never sends keystrokes, typed text, clipboard data, browser page content, passwords, screenshots, or window titles.

Current-user reporting and privacy-sensitive activity reporting default off. Future providers for active apps, windows, camera, microphone, games, calls, or recordings must be separately toggled and clearly labeled.

Local logs retain connection events, command names/results, provider failures, startup/shutdown, and versions. Tokens are never logged. Diagnostic export omits user names, application/window data, credentials, and exact local paths by default. PC Bridge has no analytics or external telemetry endpoint.
